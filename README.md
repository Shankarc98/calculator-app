# Web Calculator Application
A Calculator Application that supports basic arithmetic operations like addition, subtraction, multiplication, division, remainder.


## Live Demo
calculator-app-ecru-nu.vercel.app

## Why this Project ? 
This project was created to practice React state management, event handling, conditional logic, and calculator operations such as addition, subtraction, multiplication, division, remainder, and decimal input.

## Features
- Perform basic arithmetic operations:
   1) Addition (+)
   2) Subtraction (-)
   3) Multiplication (x)
   4) Division (/)
   5) Remainder (%)
- Decimal number input
- Interactive button-based calculator UI

## How it works ? 
The calculator keeps track of:
- the current number being entered
- the selected operator
- the previous value used in the calculation
- the calculated result

It supports building multi-digit numbers and decimal numbers, and updates the display based on the user’s input through calculator buttons.

## Installation and Setup 
1. Clone the repository
git clone https://github.com/Shankarc98/calculator-app.git
2. Move into the project folder
cd calculator-app
3. Install dependencies
npm install
4. Start the development server
npm run dev

Build for production 
npm run build
